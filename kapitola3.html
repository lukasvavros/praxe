Kapitola 3: Servomotor

Praxe – Arduino | 3. ročník

Cíle kapitoly

V této kapitole se naučíme ovládat modelářský servomotor pomocí Arduina. Servomotor se nepoužívá hlavně pro trvalé otáčení, ale pro přesné nastavení polohy.

PorozuměníVysvětlíme si, z jakých částí se servo skládá a proč dokáže držet nastavený úhel.
ZapojeníPřipojíme tři vodiče serva: napájení, zem a řídicí signál.
ProgramováníPoužijeme knihovnu Servo a nastavíme polohu osy ve stupních.

Co je servomotor

Servomotor je převodový motor, který dokáže natočit svou osu do přesně určené polohy. U běžného modelářského serva je pracovní rozsah obvykle přibližně 0 až 180°.

Požadovaný úhel se servu předává elektrickými impulsy z řídicí jednotky, například z Arduina. Servo zároveň sleduje skutečnou polohu své osy a v případě potřeby motor automaticky zapne, aby dosáhlo nastavené hodnoty.

Uvnitř serva spolupracují motor, převodovka, potenciometr a řídicí elektronika.

Uzavřená řídicí smyčka

Servo porovnává cílový a skutečný úhel. Dokud se hodnoty neshodují, upravuje polohu osy.

MotorVytváří pohyb uvnitř serva. Může být stejnosměrný nebo krokový.
PřevodovkaSnižuje otáčky motoru a zároveň zvyšuje jeho točivý moment.
PotenciometrSnímá aktuální polohu osy a poskytuje zpětnou vazbu řídicí elektronice.

Zapojení serva k Arduinu

Modelářské servo má obvykle tři vodiče. Barvy se mohou u jednotlivých výrobců lišit, ale u běžného serva SG90 se často používá hnědý vodič pro zem, červený pro napájení a oranžový pro řídicí signál.

Hnědý / černý — GNDSpolečná zem. Připojí se na pin GND Arduina.
Červený — 5 VNapájení serva. U malého serva lze pro pokusy použít 5 V z Arduina.
Oranžový / žlutý — signálŘídicí vodič. Připojí se například na digitální pin D9.
Arduino D9
→
Signál serva
→
Natočení osy

Pozor na napájení: Jedno malé servo obvykle při jednoduchém pokusu funguje z 5 V Arduina. Více serv nebo větší servo však může odebírat příliš velký proud. V takovém případě použijte samostatný zdroj a propojte jeho GND s GND Arduina.

Řízení polohy impulsy

Servo se neřídí běžnou hodnotou analogWrite(). Arduino mu pravidelně posílá řídicí impulsy. Důležitá je především délka impulzu, která určuje požadovaný úhel.

Jeden řídicí cyklus trvá přibližně 20 ms, což odpovídá frekvenci asi 50 Hz. Pro názornost použijeme rozsah 0,5 až 2,5 ms; skutečné krajní hodnoty se mohou u různých typů serv mírně lišit.

0° — krátký impuls Servo nastaví osu do krajní polohy přibližně 0°.
90° — střední impuls Servo nastaví osu přibližně do střední polohy 90°.
180° — dlouhý impuls Servo nastaví osu do druhé krajní polohy přibližně 180°.
Dobrá zpráva: Knihovna Servo vytváří impulsy automaticky. Ve vlastním programu obvykle zadáváme jen požadovaný úhel ve stupních.

Ovládání pomocí knihovny Servo

Nejprve do programu vložíme knihovnu Servo a vytvoříme objekt, který bude představovat naše servo. Funkcí attach() nastavíme řídicí pin a funkcí write() požadovaný úhel.

servo1.attach(9);   →   připojení serva na D9
servo1.write(90);   →   natočení osy na 90°
#include <Servo.h>

Servo servo1;

void setup() {
  servo1.attach(9);
}

void loop() {
  servo1.write(0);
  delay(800);

  servo1.write(90);
  delay(800);

  servo1.write(180);
  delay(800);
}

Úkol 1 – základní pohyb serva

  1. Připojte servo podle zapojení v kapitole.
  2. Nahrajte program a sledujte pohyb osy mezi úhly 0°, 90° a 180°.
  3. Vyzkoušejte změnit úhly i hodnoty funkce delay().
  4. Upravte program tak, aby se servo vracelo zpět do polohy 0°.

Potenciometr ovládá servo

Hodnota z potenciometru má rozsah 0 až 1023, zatímco servo očekává úhel přibližně 0 až 180. Pro převod mezi těmito rozsahy použijeme funkci map(). 

int hodnota = analogRead(A0);
int uhel = map(hodnota, 0, 1023, 0, 180);

servo1.write(uhel);

Funkce map() převede hodnotu z jednoho rozsahu do druhého:

map(hodnota, původní minimum, původní maximum, nové minimum, nové maximum)
#include <Servo.h>

Servo servo1;

const byte pinServo = 9;
const byte pinPotenciometr = A0;

void setup() {
  servo1.attach(pinServo);
}

void loop() {
  int hodnotaPotenciometru = analogRead(pinPotenciometr);
  int uhel = map(hodnotaPotenciometru, 0, 1023, 0, 180);

  servo1.write(uhel);
  delay(15);
}

Úkoly – servo a potenciometr

  1. Vytvořte program, který pohybuje servem plynule od 0° do 180° po jednom stupni.
  2. Po dosažení 180° nechte servo plynule vrátit zpět do polohy 0°.
  3. Připojte potenciometr na analogový vstup A0.
  4. Pomocí funkce map() nastavujte polohu serva podle natočení potenciometru.
