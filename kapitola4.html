Kapitola 4: PWM a RGB LED

Praxe – Arduino | 3. ročník

Cíle kapitoly

V této kapitole se naučíme řídit jas LED pomocí pulzně šířkové modulace a následně samostatně nastavovat červenou, zelenou a modrou složku RGB LED.

PWMPochopíme princip rychlého zapínání a vypínání digitálního výstupu.
analogWrite()Naučíme se nastavovat PWM hodnoty v rozsahu 0 až 255.
RGB LEDBudeme vytvářet různé barvy kombinací tří barevných složek.

Pulzně šířková modulace – PWM

PWM je metoda, která mění průměrný výkon rychlým zapínáním a vypínáním digitálního signálu. Poměr doby zapnutí k délce celé periody nazýváme střída neboli duty cycle.

Čím déle je signál v logické jedničce, tím vyšší je průměrný výkon a tím více LED svítí.

0 %Signál je stále vypnutý.
50 %Signál je polovinu periody zapnutý.
100 %Signál je stále zapnutý.
Důležité: PWM není skutečné analogové napětí. Arduino pouze velmi rychle přepíná pin mezi LOW a HIGH.

Funkce analogWrite()

PWM nastavujeme pomocí funkce analogWrite(). Druhý parametr je hodnota od 0 do 255.

analogWrite(pin, hodnota);
0LED nesvítí.
128LED svítí přibližně na 50 %.
255LED svítí maximálním jasem.
const byte pinLed = 9;

void setup() {
  pinMode(pinLed, OUTPUT);
}

void loop() {
  analogWrite(pinLed, 0);
  delay(1000);

  analogWrite(pinLed, 128);
  delay(1000);

  analogWrite(pinLed, 255);
  delay(1000);
}

Pozor: Na Arduino UNO podporují PWM piny označené symbolem ~: 3, 5, 6, 9, 10 a 11.

RGB LED

RGB LED kombinuje v jednom pouzdře tři LED diody: červenou (R), zelenou (G) a modrou (B).

Každou složku lze ovládat samostatně. Kombinací jejich jasů můžeme vytvořit mnoho různých barev.

RGB LED má čtyři vývody. Tři patří jednotlivým barvám a jeden je společný. Podle typu jde o společnou katodu nebo společnou anodu.

Společná katodaSpolečný vývod se připojí na GND. Vyšší PWM znamená vyšší jas.
Společná anodaSpolečný vývod se připojí na 5 V a používá se obrácená logika.
PolaritaPřed zapojením ověřte typ a pořadí vývodů konkrétní součástky.

V této kapitole používáme RGB LED se společnou katodou. Každá barevná složka musí mít vlastní sériový rezistor.

Zapojení RGB LED k Arduinu

Použijeme PWM piny D9, D10 a D11. Každou barevnou složku připojíme přes samostatný rezistor 220 až 330 Ω.

R → D9Červená složka přes rezistor.
G → D10Zelená složka přes rezistor.
B → D11Modrá složka přes rezistor.
D9, D10, D11
→
3× rezistor 220–330 Ω
→
RGB LED, společná katoda → GND
const byte pinCervena = 9;
const byte pinZelena = 10;
const byte pinModra = 11;

void setup() {
  pinMode(pinCervena, OUTPUT);
  pinMode(pinZelena, OUTPUT);
  pinMode(pinModra, OUTPUT);
}

void loop() {
  analogWrite(pinCervena, 255);
  analogWrite(pinZelena, 0);
  analogWrite(pinModra, 0);
}

Míchání barev

Výsledná barva vzniká kombinací hodnot R, G a B. Každá složka může mít hodnotu od 0 do 255.

Barva Ukázka R G B
Červená 255 0 0
Zelená 0 255 0
Modrá 0 0 255
Žlutá 255 255 0
Azurová 0 255 255
Purpurová 255 0 255
Růžová 255 40 120
Bílá 255 255 255
Skutečný odstín závisí na konkrétní LED, použitých rezistorech a citlivosti lidského oka.

Úkol 1 – tři úrovně jasu

Zhasnuto, 50 % a 100 %

Vyberte jednu barevnou složku RGB LED. Jednotlivé podúkoly na sebe navazují.

  1. Nastavte hodnotu 0.
  2. Po jedné sekundě nastavte hodnotu 128.
  3. Po další sekundě nastavte hodnotu 255.
  4. Po další sekundě LED zhasněte.
  5. Celý cyklus nechte stále opakovat.
Rozšiřující úkol: Vyzkoušejte program pro všechny tři barevné složky.

Úkol 2 – plynulá změna jasu

Rozsvícení a zhasnutí pomocí cyklu

  1. Pomocí cyklu for měňte jas od 0 do 255.
  2. V každém průchodu použijte analogWrite().
  3. Přidejte krátkou prodlevu.
  4. Druhým cyklem snižte jas zpět na 0.
  5. Program nechte stále opakovat.
Rozšiřující úkol: Změňte krok cyklu na 5 nebo 10 a porovnejte plynulost.

Potenciometr ovládá jas

analogRead() vrací 0 až 1023, zatímco analogWrite() očekává 0 až 255. Rozsah převedeme funkcí map(). 

int hodnotaPotenciometru = analogRead(A0);
int jas = map(hodnotaPotenciometru, 0, 1023, 0, 255);
analogWrite(pinCervena, jas);

Úkol 3 – řízení jasu potenciometrem

  1. Připojte potenciometr na A0.
  2. Načtěte jeho hodnotu funkcí analogRead().
  3. Převeďte ji funkcí map() na 0 až 255.
  4. Nastavujte jas jedné barevné složky.
  5. Vypisujte původní i převedenou hodnotu do sériového monitoru.
Rozšiřující úkol: Jednu barvu zesilujte a druhou současně zeslabujte.

Úkol 4 – vlastní barvy

Míchání barevných složek

  1. Nastavte růžovou barvu.
  2. Nastavte žlutou barvu.
  3. Nastavte azurovou barvu.
  4. Nastavte bílou barvu.
  5. Vytvořte vlastní odstín a zapište jeho hodnoty R, G a B.
Rozšiřující úkol: Vytvořte funkci nastavBarvu(r, g, b).

Úkol 5 – plynulý přechod barev

Modrá → zelená → modrá

  1. Nastavte počáteční modrou barvu.
  2. V cyklu zvyšujte zelenou složku.
  3. Ve stejném cyklu snižujte modrou složku.
  4. Po dosažení zelené vytvořte opačný přechod.
  5. Rychlost upravte pomocí delay().
Rozšiřující úkol: Přidejte přechody zelená → červená a červená → modrá.

Úkol 6 – tři potenciometry

Ruční mixáž RGB barev

Každý potenciometr bude nastavovat jednu barevnou složku.

  • R → A0
  • G → A1
  • B → A2
  1. Načtěte hodnoty všech tří potenciometrů.
  2. Každou převeďte na rozsah 0 až 255.
  3. Výsledky odešlete na příslušné RGB výstupy.
  4. Vypisujte R, G a B do sériového monitoru.
  5. Namíchejte růžovou, oranžovou, fialovou a bílou.
Rozšiřující úkol: Přidejte tlačítko, které vypíše aktuální hodnoty jako uloženou barvu.
